#  Pro Digital Clock

A modern, feature-rich and beautifully designed **Digital Clock Web App** built using **HTML, CSS, and JavaScript**.
Includes multiple utilities like **Alarm**, **Stopwatch**, **Timer**, **World Clock**, dynamic **Themes**, **Dark Mode**, **Voice Time Announcement**, and **Fullscreen mode** — all in one clean interface.

##  Features

###  1. Flip Digital Clock

* Smooth animated flip digits
* 12-hour format with AM/PM
* Real-time date display
* Customizable color themes

###  2. Voice Time Announcement

* Speaks the current date & time
* Uses Web Speech API
* Indian English (en-IN) voice

###  3. Alarm

* Set any alarm time
* Auto-rings with built-in sound
* Visual alert on trigger

### ⏱ 4. Stopwatch

* Start / Stop / Reset
* Accurate counting
* Clean digital display

###  5. Countdown Timer

* Enter time in seconds
* Live countdown
* Displays “Finished!” at end
* Stop anytime

###  6. World Clock

Shows live time for:

* India (IST)
* New York
* London
* Tokyo
* Sydney

###  7. Themes + Dark Mode

Built-in theme presets:

* Default
* Ocean Blue
* Sunset Orange
* Galaxy Purple
* Neon Dark
* Minimal White

Also includes a modern **Dark Mode toggle**.

###  8. Fullscreen Mode

* Enter immersive fullscreen clock
* Exit with one click

###  9. Tab Navigation

Switch between all features smoothly without reloading the page.

##  How to Run

1. Download or clone the repository:

   ```bash

   ```
2. Open **index.html** in any modern browser.
3. That’s it — no installation required!
